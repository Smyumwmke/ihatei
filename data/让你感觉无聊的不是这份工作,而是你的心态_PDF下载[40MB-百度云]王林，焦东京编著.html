总目录 **章 重视无聊感觉，远离工作中的消极心态 无聊是一种情绪，是心理上的一种状态。在这个快节奏的时代，我们都会因为工作、情感、学习等问题而出现心理压力，也会出现无聊状态。无聊并不可怕，可怕的是一直无聊的消极心态。所以，我们要学着改变无聊心态，不能被无聊控制，要去分析自己，挑战自己，戒掉无聊的“毒瘾”。 1. 无聊是一种本能和常态 2. 无聊并不可怕，可怕的是一直无聊的心态 3. 不要得过且过，对工作敷衍了事 4. 走出无聊阴影，树立人生目标和梦想 5. 摒弃打工心态，工作是为了老板更是为了自己 6. 搞清楚自己是真的厌倦工作，还是心态出了问题 第二章 塑造健康心态，你的心态变了世界就变了 健康不仅是没有疾病，更是一种个体在身体上、精神上和社会上的完全安宁状态。生命需要阳光，人的心灵也是一样。积极健康的心态，就是心灵的营养。这样的心灵，能吸引财富、成功、快乐。消极的心态，却是心灵的疾病和垃圾。这样的心灵，不仅排斥财富、成功、快乐，甚至会夺走生活中已有的一切。 1. 乐观自信，改变消极自卑的心态 2. 摒弃怠慢工作的心态，跳出“无聊”陷阱 3. 心态阳光一些，激发正能量 4. 与其抱怨自己工资太低，不如努力工作 5. 拒绝平庸，平凡岗位也能做出成就 6. 时刻准备，勇于抓住每次机会 第三章 学会心理调控，摆脱心态上的无聊枷锁 当今社会，激烈的竞争和物质诱惑无时无刻不在扰动着我们的心灵，因此我们常常感到无聊、焦躁、不安、愤怒。人们也尝试用物质的方式进行调整，但往往并不能奏效。所以，我们要及时地发现心理上存在的问题，尽早对其进行调适。 1. 呵护心理健康，提高心理适应能力 2. 摆正心态，对工作负责充满热爱 3. 感恩工作，快乐地接受工作的恩赐 4. 静下心来，放慢工作和生活的节奏 5. 改变孤独心态，融入职场团队 6. 警醒自己，学会心理调整的技巧 第四章 排除情绪病毒，心情舒畅才能避免无聊苦闷 在工作生活中，一个人产生无聊心理，就会对日复一日习以为常的生活失去兴趣，对周围美好的东西视若无睹。情绪对我们的影响，我们往往低估了。我们在工作中要学会体察自己的情绪，善于管理好自己的情绪。 1. 无聊是一种情绪病毒 2. 管好情绪是心灵健康的根基 3. 控制暴躁脾气，浇灭心头的愤怒之火 4. 向亲朋好友倾诉，疏散郁闷情绪 5. 避免抱怨，与其抱怨不如改变 6. 淡定从容，心平气和做自己 第五章 克制无聊疲惫，点燃激情向职业倦怠开战 一个人长期从事某种职业，在日复一日的工作中，渐渐就会产生疲惫、困乏，甚至厌倦的心理，在工作中难以提起兴致，打不起精神，这直接影响到工作准备状态，然后又反作用于工作，导致工作状态恶化。我们要唤醒积极的工作态度，像热爱生命一样激情工作，才能克服无聊疲惫，才能把工作做到位，有一番作为。 1. 职业倦怠：生命中不能承受之重 2. 注入激情，永远保持一颗热忱的心 3. 乐业敬业，享受工作的过程 4. 不怕困难，把工作难关当作挑战 5. 劳逸结合，适当休息化解心理疲劳 6. 检视自我，换工作不如换心情 第六章 培养兴趣爱好，让生活与工作结合提升新鲜感 工作没有成就感，是因为没有将工作兴趣化，能不能从从事的工作中感受到乐趣，归根到底是一个心态问题。当你工作兴趣越高、决心越大，你的工作效率也就会越高。当你把工作当作一件开心和有满足感的事，就不会感到单调和无聊。 1. 找到工作兴趣点，干一行爱一行 2. 珍惜工作，高高兴兴把工作做好 3. 丰富休闲方式，平衡工作与生活 4. 自我放松，多参加体育活动 5. 在游山玩水中化解无聊 6. 音乐平心气，多听益健康 第七章 拒绝工作拖延，不要让时光虚度在无聊中 在这个世界上，没有别的什么习惯比拖延更为有害；更没有别的什么习惯比拖延更能使人懈怠和减弱人们做事的能力。因此，我们要重视起来，改变拖延的恶习，珍惜时间，高效工作，提升核心竞争力，赢得工作中的辉煌。 1. 高效管理时间，珍惜每分每秒 2. 计划周详，让工作不再盲目杂乱 3. 找对工作方法，提升工作效率 4. 日事日清，不要被拖延捆住了手脚 5. 拒绝借口，自觉守纪律严执行 6. 不要让私人琐事干扰工作 第八章 保持学习欲望，不断提高自己的工作实力 一个愿意通过学习来提升自己能力的人，*终会获得职位上的升迁和事业上的成功。学习的目的不是为了追求高学历，而是不断提高自己的能力。一个现在有能力的人，无论他是博士、硕士，还是高级工程师，如果不注重学习，也会落后，变成一个“能力平平”的人。而一个暂时能力不是很强的人，只要坚持学习，善于学习，就一定会成为一个能力出众的人。 1. 完善自我，用业余时间充电 2. 没有“金刚钻”，别揽“瓷器活” 3. 主动学习，重“学历”更重“学力” 4. 虚心学习工作中的新知识、新技术 5. 学习竞争对手的经验和长处 6. 坚持学习，每天进步一点点 第九章 勇于创新思考，敢想敢做，你的工作就不会无聊 当今快节奏、多变化、竞争日趋激烈的社会，对每一个员工都提出了前所未有的创新要求。“守旧者死，创新者生。”具备创新能力的优秀员工，成了企业亟需的宝贵财富，他们永远不会有失业之虞。因此，在工作中，员工要大力发扬岗位创新精神，勇于创新思考，这样你的工作就不会无聊。 1. 每一个岗位都是创新的舞台 2. 激活创新细胞，带着思想去工作 3. 打破旧框框，不墨守成规 4. 灵活变通，学会另辟新路径 5. 敢冒风险，大胆尝试新事物 6. 勇于创新，成为不可替代的人 第十章 心无旁骛工作，全心全意才能排除无聊干扰 现代社会，机会多多，选择多多，诱惑也就更多。人们普遍呈现浮躁的心态。在这种心态的驱使之下，人们容易见异思迁，走上迷途。对此，我们要专注地做事，专注可以使我们静下心来，心无旁骛，一心一意，把工作做完、做好。倘若我们好高骛远，见异思迁，心浮气躁，什么都想抓，*终则会一无所获。 1. 脚踏实地，不好高骛远 2. 专注工作，摒弃三心二意 3. 注重细节，将小事情做好 4. 认真负责，杜绝粗心大意 5. 把握现在，马上去做 6. 勤勤恳恳，一分付出一分收获
